
# Bitcoin Sentiment vs Trader Performance (Data Analyst Project)

This project analyzes how Bitcoin market sentiment (Fear & Greed Index) affects trader profitability using historical Hyperliquid trading data.

## What was done
- Cleaned and aligned two datasets by date
- Merged trader PnL with market sentiment
- Compared performance across Fear, Neutral, and Greed periods
- Visualized key insights

## Files
- data/: cleaned and merged datasets
- notebooks/: analysis notebook
- src/: python scripts
- results/: charts and outputs
